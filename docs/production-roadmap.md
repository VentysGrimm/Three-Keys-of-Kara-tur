# Production Roadmap

This roadmap converts the written Kara-Tur solo campaign into a complete Foundry VTT D&D 5e adventure module.

## Phase 1: Foundry Skeleton

- Add `module.json` manifest.
- Add script, style, language, and documentation folders.
- Add compendium pack placeholders.
- Confirm the module can be installed into Foundry.

## Phase 2: Adventure Blueprint

- Define the complete Adventure document structure.
- Create journal folders for Acts I–IV.
- Establish naming conventions for Actors, Items, Scenes, Roll Tables, Playlists, and Macros.
- Convert the solo-player companion model into a Foundry reference system.

## Phase 3: Act I Vertical Slice

Build a fully playable version of **Act I — The Temple Lie**.

Deliverables:

- Start Here journal
- Act I GM overview
- Act I scene journals
- Temple of the Dawn Chain overview
- Initial companion NPC Actors
- Initial cultist and temple threat Actors
- Starter Items and sealing references
- Roll tables for temple omens and early missions
- Placeholder Scenes

## Phase 4: Full Campaign Conversion

- Convert Act II: Pilgrimage of the Three Keys.
- Convert Act III: The Breaking Pilgrimage.
- Convert Act IV: The Fox Behind the Human Face.
- Add disaster cults, avatars, Atsumori, sealing arts, and ending variants.

## Phase 5: Foundry Integration

- Create compendium packs.
- Create or import the Adventure document.
- Link journals to Actors, Items, Scenes, and Roll Tables.
- Add token placements, map pins, scene notes, and encounter setup.

## Phase 6: Testing

- Import into a clean Foundry world.
- Verify module activation.
- Verify all compendium packs are visible.
- Verify Adventure import.
- Check links, paths, images, scene data, and D&D 5e item/actor compatibility.

## Phase 7: Polish

- Replace placeholder art.
- Add battle maps, lighting, walls, sound, and token art.
- Add macros for disaster clocks, companion reactions, and pilgrimage events.
- Finalize licensing and release packaging.
